# One Ummah - Islamic Social Media Platform

An Instagram-style Islamic social network with prayer times, Adhan, and community features.

## Features
- Instagram-style camera (video/photo)
- Prayer times for worldwide cities
- 8 Adhan reciters with full audio
- Prayer notifications
- Friend system
- Direct messaging
- Posts, comments, likes, shares
