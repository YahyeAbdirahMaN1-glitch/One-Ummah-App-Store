## ✨ Features

### 📸 Instagram-Style Camera
- Video & Photo modes
- Littles (short vertical videos)
- Length (longer horizontal videos)
- Front/back camera flip

### 🕋 Prayer Times & Adhan
- Worldwide prayer times (search by city/country)
- 8 authentic Adhan reciters
- Customizable prayer notifications
- Background notifications

### 👥 Community Features
- Social feed (posts, comments, likes, shares, reposts)
- Direct messaging with read receipts
- Friend system with location-based suggestions
- User profiles with authentication
